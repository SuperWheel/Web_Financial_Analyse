# Spec 003: 公开上市公司财报导入

## ADDED 需求

### ADDED-1：上传与任务

- **要求**：支持上传 PDF（一期）；创建 `ImportJob`，文件存 `data/imports/`。
- **验收**：`POST /api/imports/filings` 返回 job id 与初始状态。

### ADDED-2：自动识别三大报表

- **要求**：对 A 股 CAS 数字 PDF，定位合并口径资产负债表/利润表/现金流量表。
- **要求**：区分合并表与母公司表，默认合并。
- **验收**：科沃斯、影石创新样本能识别三表标题与主要行项目。

### ADDED-3：科目映射与单位

- **要求**：将源表标签映射到系统字段；支持元/千元/百万元换算为元。
- **要求**：未映射行可列出，不静默丢弃信息（可在草稿中查看）。
- **验收**：核心字段（货币资金、资产总计、营业收入、净利润、经营活动现金流量净额等）在 CAS 样本上映射正确。

### ADDED-4：人审确认入库

- **要求**：提供预览与映射修正；用户确认后写入企业与三表。
- **要求**：同企业同报告期冲突时可选跳过或覆盖。
- **验收**：commit 后 `GET` 报表接口可见数据；未确认前不写业务表。

### ADDED-5：难解析样本降级

- **要求**：对无法可靠抽取的 PDF（如 CID 字体），任务失败或标记需人工/其他源，并给出可读提示。
- **验收**：小米样本不产生错误数值入库。

### ADDED-6（扩展）：港股/US GAAP 适配

- **要求**：哔哩哔哩类繁体 + 千元 + 多列年份可部分映射。
- **验收**：至少完成公司识别、期间、单位换算与核心损益/现金科目映射；允许部分 unmapped。

## 非本变更范围

- 实时行情、自动爬全市场年报。
- OCR 全自动（可列二期）。
- 完整准则转换（US GAAP→CAS 重述）。
- 附注级明细抽取。
