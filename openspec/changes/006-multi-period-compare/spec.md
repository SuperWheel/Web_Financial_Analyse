# Spec 006: 科目级多期对比

## 需求

### R1 可对比期间列表

- 给定企业，返回该企业各报告期是否具备资产负债表 / 利润表 / 现金流量表。
- 排序：新→旧（便于前端默认勾选最近几期）。

### R2 多期科目矩阵

- 入参：`company_id`、`statement_type`（`balance`|`income`|`cashflow`）、`period_type`（`annual`|`quarterly`）、可选 `years`（逗号分隔年份过滤）。
- 仅同 `period_type` 内对比，不混比年报与季报。
- 输出按科目分组：每科目在各期的金额；相对**序列上一期**的变动额与变动率；结构占比（见 R3）。
- 缺表或缺科目时该期值为 `null`，不抛错。

### R3 结构占比

| 报表 | 分母 |
|------|------|
| balance | `total_assets`（资产总计） |
| income | `operating_revenue`（营业收入） |
| cashflow | 不计算结构（`structure_pcts` 全 `null`） |

分母为 0 或缺失时该期结构为 `null`。

### R4 前端

- 路由 `/compare`，侧栏「多期对比」。
- 选择企业、报表类型、年报/季报、多选期间（默认最近最多 5 期）。
- 表格：科目 × 各期金额；可切换显示结构% / 同比%。
- 趋势：勾选或点击科目后折线展示多期金额。
