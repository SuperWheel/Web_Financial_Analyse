# 项目背景：Web_Financial_Analyse 财务分析系统

## 一句话简介

一个轻量级的**企业财务报表分析系统**，支持三大财务报表（资产负债表、利润表、现金流量表）的录入、存储、公开年报导入、财务比率计算与多期对比，辅助财务分析与决策。

## 当前版本

- **已发布**：**v1.0.0**（git tag `v1.0.0`）
- **形态**：单机本地，无认证，SQLite 落地于 `data/finance.db`
- **主路径**：A 股 CAS 年报（PDF / Excel / 巨潮在线拉取）+ 手工录入

## 业务范围

- **核心场景**：企业财务报表分析（非个人记账、非投资行情）。
- **v1.0 已覆盖**：
  - 企业信息管理（公司档案）。
  - 三大报表数据录入与存储（年报 / 季报；CAS-simplified-v2 科目 + L0 披露明细）。
  - 公开年报导入：PDF 识别映射 + 人审入库；Excel 模板导入 / 导出（含财务比率 sheet）。
  - 年报在线拉取：PDF 直链 + 巨潮（代码或公司名称检索）。
  - 财务比率分析（流动比率、速动比率、ROE、毛利率、净利率、资产负债率等；动态计算不落库）。
  - 科目级多期对比与趋势分析（金额 / 环比 / 结构占比）。

## 技术定位

- **轻量级**：单机本地运行，无需复杂部署与认证。
- **技术栈**：FastAPI（后端）+ Vue3（前端）+ SQLite（存储）+ pandas（计算）+ openpyxl / pdfplumber / httpx。
- **目标用户**：个人财务分析者 / 小型分析团队。

## 非目标（v1.x 不做）

- 不做多用户与权限系统。
- 不接入实时行情 / 股票报价数据源。
- 不做预测性建模 / 机器学习。
- 不做云端部署。
- 不自动无人确认入库（财务数字必须人审）。
